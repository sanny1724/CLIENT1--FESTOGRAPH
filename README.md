## üöÄ Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm (Node Package Manager)

### Installation
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sanny1724/CLIENT1--FESTOGRAPH.git
    cd festograph
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

## üì¶ Deployment
This project is deployment-ready for **Netlify**.

1.  Buffer build command: `npm run build`
2.  Publish directory: `dist`
3.  **Note**: A `netlify.toml` file is included to handle routing automatically.

For a manual deploy, simply run `npm run build` and drag the `dist` folder to [Netlify Drop](https://app.netlify.com/drop).
